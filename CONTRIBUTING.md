# Contributing

Thanks for improving the agent library.

Before opening a pull request:

1. Keep agent definitions generic and reusable.
2. Remove proprietary product details, private names, customer references, credentials, source snippets, and screenshots.
3. Add or update the human-readable agent card in `agents/`.
4. Add or update the machine-readable entry in `agents/catalog.yaml`.
5. Run the public-safe scan in `docs/no-proprietary-data.md`.

Good contributions make agents more useful without making assumptions about a specific company, product, stack, or private workflow.
